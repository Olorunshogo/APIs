Jottings on Application Programming Interface (API)

Representational State Transfer (ReST) APIs are useful for ETL (extract, transform, load) and data integration workflow.

    Advantages include:
  • Simple and Standardized: No format
  • Sacalable and stateless: Easy modifications even when bigger.
  • High performance because of catching
    Likely actions = HTTP methods include:
  • Create = POST
  • Read = GET
  • Update = PUT
  • Delete = DELETE
